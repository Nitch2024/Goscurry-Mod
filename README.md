# Build Instructions with Visual Studio
1. Copy the following DLLs (from Goscurry\Goscurry_Data\Managed) into the same folder as Plugin.cs
*Assembly-CSharp.dll
*DOTween.dll
*GameCore.dll
*Goscurry.Assemblies.dll
*HO2DToolkit.dll
*UnityEngine.dll
3. Add 0Harmony.dll (from https://github.com/pardeike/Harmony) into the same folder as Plugin.cs
4. Open Goscurry.sln and build
